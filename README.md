## 컨벤션
### 1) 코딩 컨벤션
- prettier 사용 X
- 들여쓰기 2개
- `reset.css`는 http://meyerweb.com/eric/tools/css/reset/ 사용
- z-index는 10단위로
- class명은 형태 + 의미 + 상태
- [toast-ko_HTMLCSS](https://ui.toast.com/fe-guide/ko_HTMLCSS), [toast-ko_CODING-CONVENTION](https://ui.toast.com/fe-guide/ko_CODING-CONVENTION)을 주로 따름
